# Impact Report: ממשק ניהול פעילויות וקבוצות

תאריך: 2026-06-11  
סיווג שינוי: feature  
רמת סיכון: High

## מה משתנה

נוסף ממשק ניהול מלא לפעילויות וקבוצות:

- API ניהול לפעילויות.
- API ניהול לקבוצות.
- עמוד SPA `/admin/activities`.
- קישור ניווט חדש בתפריט הניהול.

## השפעה לפי תחום

Database:

- אין שינוי schema.
- ניהול רשומות `activities` ו-`groups` מתבצע ללא מחיקה פיזית.
- `publish_on_site` ו-`status` ממשיכים לשלוט בחשיפה הציבורית.

API:

- נוספו נתיבי admin ליצירה ועריכה של פעילויות וקבוצות.
- הנתיבים מוגנים על ידי `requireAdmin`.

Components:

- נוסף `admin-activities-page`.
- עודכן `sidebar`.

QA:

- נדרשת בדיקת build, context, db:init ואימות שאין אחסון JWT בצד לקוח.

## סיכונים

- פרסום פעילות או קבוצה בטעות באתר.
- שבירת נתיבי public עקב שינוי חוזי ניהול.
- שיוך מדריך לא פעיל לקבוצה.

## הפחתת סיכונים

- public APIs ממשיכים לסנן לפי `publish_on_site=true` ו-`status=active`.
- validation מוודא מדריך פעיל לפני שיוך.
- אין מחיקה פיזית; ארכוב נעשה דרך `status`.

## סטטוס

מיושם ואומת באמצעות `context:check`, `npm test` ו-`db:init`.
