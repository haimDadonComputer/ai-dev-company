# שכבת API

שכבת השרת מספקת Authentication למנהל, הגדרות אתר כלליות וניהול מדיה.

## Authentication

- האימות מבוסס JWT ב־HttpOnly Cookie בשם `admin_session`.
- רק משתמש פעיל בעל role מסוג `admin` יכול להתחבר.
- מנהל עם `mustChangePassword=true` יכול לגשת רק ל־`me`,
  `change-password` ו־`logout`.
- התחברות מוגבלת לחמישה ניסיונות כושלים בחלון של 15 דקות לפי IP ושם משתמש.

## מדיה

- העלאה נעשית כ־`application/octet-stream`.
- שם הקובץ מתקבל ב־`X-File-Name` וטקסט חלופי ב־`X-Alt-Text`.
- הגודל המרבי הוא 5MB ונבדק גם לפי הזרם בפועל.
- רק PNG, JPEG ו־WebP שמזוהים לפי magic bytes מתקבלים.
- נתיב האחסון הפנימי אינו מוחזר באף API.
- מחיקה היא רכה ונחסמת כאשר המדיה מקושרת להגדרות האתר.

## הגדרות

- API הניהול מחזיר גם `updatedByUserId`.
- API הציבור אינו מחזיר שדה זה ומייצר כתובות `/api/media/:id`.
- מזהי מדיה נבדקים מול נכסים קיימים ופעילים לפני שמירת ההגדרות.
