# Roadmap

## MVP

- משתמש `admin` התחלתי עם סיסמה זמנית מגובבת.
- התחברות, התנתקות והחלפת סיסמה ראשונית.
- מעטפת ממשק ניהול SPA, RTL ורספונסיבית.
- עמוד `כללי` לניהול פרטי האתר.
- העלאת תמונות וקבצים מאובטחת עד 5MB.
- בחירת לוגו ותמונות מתוך המדיה שהועלתה.
- API ציבורי מצומצם לקריאת הגדרות האתר.

## Version 1.1

- ספריית מדיה עם חיפוש, סינון ומחיקה רכה.
- היסטוריית שינויים להגדרות.
- ניהול משתמשי מנהלה נוספים לאחר אפיון הרשאות.

## Version 2.0

- התאמת התשתית לפרויקטים מרובים באמצעות תבניות הגדרה.
- אחסון קבצים חיצוני כאפשרות, בכפוף לאישור ארכיטקטוני.
- הרשאות מפורטות לפי מודולים ופעולות.
