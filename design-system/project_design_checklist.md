# Project Design Checklist

לפני פיתוח ממשק בפרויקט חדש:

- [ ] הוגדר קהל יעד.
- [ ] הוגדר אופי מותג בשלוש עד חמש מילים.
- [ ] נבחר קונספט חזותי שאינו מבוסס על פרויקט קודם.
- [ ] נבחר navigation pattern.
- [ ] נבחרו טיפוגרפיה, צפיפות ושפת צורות.
- [ ] הוגדרו צבעים נגישים עם contrast מתאים.
- [ ] הוחלף `profileId`.
- [ ] `status` שונה ל־`custom`.
- [ ] `referenceOnly` שונה ל־`false`.
- [ ] לפחות ארבעה ממדי שונות תועדו.
- [ ] `npm run design:check` עבר ללא override.
- [ ] בוצעה בדיקת mobile ו־desktop.
- [ ] צולמו מסכי מפתח לצורך Visual QA.
